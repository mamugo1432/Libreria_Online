<h1>Librería Online</h1>

<section>
  <article>
    <h2>Descripción</h2>
    <p>El proyecto consiste en una librería online en la los usuarios se podrán marcar sus libros favoritos.</p>
    <p>Este está construido usando Vite y JavaScript Vanilla para realizar una aplicación usando el patrón de diseño SPA(Single Page Application). Para la creación de la API se ha ultilizado Supabase y para el desplieque se ha utilizado Vercel.</p>
  </article>
  <article>
    <h2>Rutas disponibles</h2>
    <p>La rutas disponibles en la aplicación web son: </p>
    <ul>
      <li>Login -> /#/login</li>
      <li>Books -> /#/</li>
      <li>Book Details -> /#/books/:id</li>
      <li>Favorites -> /#/favorites</li>
    </ul>
  </article>
    <article>
    <h2>Estado Almacenado en LocalStorage</h2>
    <p>LocalStorage es utilizado en este proyecto para guardar la lista de los libros favoritos del usuario. Esta lista está compuesta por los id de aquellos libros que son marcados como favoritos.</p>
  </article>
